== Description ==

The calendar plugin is a simple & elegant widget designed for all wordpress sites/blogs.

After plugin activation you will be automatically redirected to your widgets page so you can configure the The Calendar widget in your sidebar.
That simple!

If you need help with the plugin or you want to report a bug please refer to http://seo.uk.net/the-calendar/

== Installation ==

1. Upload the zip to 'plugins' directory or from WP Admin -> Plugins -> and search for The Calendar.
1. Activate the plugin
1. You will be automatically redirected to your widgets page so you can configure the The Calendar widget in your sidebar.

The Calendar Widget does not have a settings section on Admin page.

If you need help with the plugin or you want to report a bug please refer to http://seo.uk.net/the-calendar/
